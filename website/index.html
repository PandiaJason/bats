AI in Decision Path

Three-tier deterministic verdicts.

BLOCK

Hard Deny

136 pre-compiled regex rules covering filesystem destruction, SQL injection, reverse shells, credential theft, cloud resource deletion, and privilege escalation. Immediate rejection, no override.

CHALLENGE

Halt and Ask

47 rules for risky-but-legitimate operations: force pushes, sudo, package installs, Docker runs. The agent must stop and get explicit human re-approval before proceeding.

ALLOW

Safe to Execute

No dangerous pattern matched. The action is approved and logged in the hash-chained WAL with fsync durability. Optional async AI annotation for metadata enrichment.

Production-grade infrastructure.

Hash-Chained WAL

Every decision -- BLOCK, CHALLENGE, ALLOW -- is recorded in a SHA-256 hash-chained Write-Ahead Log with fsync durability. Any tampering breaks the chain. JSON and CSV export for compliance.

Never Delegate

No AI, LLM, or probabilistic system participates in safety decisions. The policy engine is pure regex pattern matching -- fully auditable, fully deterministic, fully reproducible.

Defense in Depth

mTLS transport, crypto/rand nonces with replay protection, per-IP rate limiting, request body size limits, graceful shutdown with connection draining, structured JSON logging.

Drop-in MCP integration.

Works with Claude Code, Antigravity, or any MCP-compatible AI agent. Three tools exposed: validate_action, check_health, get_audit_log.
