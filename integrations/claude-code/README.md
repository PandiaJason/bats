# BATS MCP Integration for Claude Code / Antigravity

A Model Context Protocol (MCP) server that gives Claude Code, Antigravity, or any MCP-compatible AI assistant access to BATS safety validation.

Every proposed action is routed through your BATS node's heuristic safety gate and Byzantine consensus layer before execution.

## Quick Setup

### 1. Build the Binary

```bash
cd integrations/claude-code
go build -o bats-mcp mcp_server.go
```

Or install globally:
```bash
go install bats/integrations/claude-code@latest
```

### 2. Start a BATS Node

```bash
# Standalone (single node, local heuristics)
go run cmd/node/main.go node1 8001

# Or Docker cluster
docker compose up
```

### 3. Configure Claude Code

Add to your Claude Code MCP settings (`~/.claude/claude_desktop_config.json` or project-level):

```json
{
  "mcpServers": {
    "bats-safety": {
      "command": "/path/to/bats-mcp",
      "args": ["--node", "localhost:8001", "--insecure"]
    }
  }
}
```

> Remove `--insecure` in production. Use proper mTLS certs generated by `scripts/gen-certs.sh`.

### 4. Configure Antigravity

For Antigravity, add to your workspace MCP config:

```json
{
  "mcpServers": {
    "bats-safety": {
      "command": "/path/to/bats-mcp",
      "args": ["--node", "localhost:8001", "--insecure"]
    }
  }
}
```

## Available Tools

### `validate_action`

Validates any proposed action through the BATS safety pipeline.

```
Action: "rm -rf /tmp/data"
→ 🚫 BLOCKED (reason: Destructive filesystem operation detected)

Action: "SELECT * FROM users WHERE id = 5"
→ ✅ APPROVED (fast-path, confidence: 0.98)

Action: "UPDATE users SET role = 'admin'"  
→ ✅ APPROVED (PBFT consensus, confidence: 0.72)
```

### `check_health`

Returns the current status of the connected BATS node.

### `get_audit_log`

Retrieves recent entries from the tamper-evident WAL (Write-Ahead Log).

## Architecture

```
Claude Code / Antigravity
       │
       │ JSON-RPC 2.0 (stdio)
       ▼
  ┌────────────────────┐
  │  bats-mcp          │
  │  (this binary)     │
  └────────┬───────────┘
           │ HTTPS
           ▼
  ┌────────────────────┐
  │  BATS Node         │
  │  :8001              │
  │  AI Gate + PBFT    │
  └────────────────────┘
```

## Flags

| Flag | Description | Default |
|:---|:---|:---|
| `--node` | BATS node address | `localhost:8001` |
| `--insecure` | Skip TLS verification (dev only) | `false` |
